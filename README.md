# AutoDream - Landing Catalog Next Level 🚀

Esta es una landing de otro nivel para **AutoDream Compraventa**, diseñada con **React 19**, **Tailwind CSS v4** y **Framer Motion**.

## Características Premium:
- **Estética Dark/Luxury**: Fondo negro absoluto con acentos púrpura lavanda (marca original).
- **Tipografía de Alto Impacto**: Orbitron para un look tecnológico/automotriz y Inter para legibilidad.
- **Micro-animaciones**: Transiciones suaves al hacer scroll y hover effects "locos".
- **Responsive Design**: Optimizada para celulares, donde ocurre la mayoría de la navegación.
- **CTA Directo a WhatsApp**: El botón principal está diseñado para generar contactos inmediatos.

## Cómo ejecutar el proyecto:

1. Asegúrate de tener **Node.js** instalado.
2. Abre una terminal en esta carpeta.
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
5. Abre `http://localhost:5173` en tu navegador.

## Estructura:
- `src/App.tsx`: Contiene toda la lógica y secciones de la landing.
- `src/index.css`: Configuración de Tailwind v4 y variables de diseño.
- `vite.config.ts`: Configuración con el plugin oficial de Tailwind v4.
